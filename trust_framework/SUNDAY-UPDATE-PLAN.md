## Contingency: If Things Go Wrong

### Technical Failure (Coordinator Down)
> "Coordinator experiencing issues. Agents attesting locally. Data will be aggregated post-recovery."
>
> Fallback: Agents save attestations locally. Reconcile later.

### Agent Dropout (>5 agents)
> "5 agents dropped due to [reason]. Continuing with 27. Minimum viable for consensus."
>
> Fallback: Lower threshold or reschedule.

### Consensus Impossible (Wide Disagreement)
> "Attestations show wide disagreement on [specific claim]. Investigation ongoing."
>
> Fallback: Publish raw data. Community analysis. v1.1 fixes.

### Economic Layer Failure ($ALPHA integration broken)
> "$ALPHA settlement delayed. Attestations recorded. Settlement will follow."
>
> Fallback: Manual settlement by @tudou_web3. Continue with attestations.
