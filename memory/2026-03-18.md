## 2026-03-18 22:15 - Build Timeout Fixes

### Issues Fixed
- Vercel build failing with 60s timeout on /page static generation
- API calls to Supabase timing out during build

### Solutions Applied
1. Home page: Added export const dynamic = 'force-dynamic'
   - Forces server-side rendering at request time
   - Prevents API call blocking during static generation

2. API client: Added 5-second fetch timeout with AbortController
   - Prevents indefinite hangs when Supabase unreachable
   - Clean error handling for timeouts

### Root Cause
Next.js static generation calls getLeaderboard() which hits Supabase. Supabase connection fails during Vercel build, causing 60s hang.

### Pattern
Use export const dynamic = 'force-dynamic' for pages that fetch external APIs during render.
